# Sistema de Gestión de Pedidos - Delivery

### Software de administración de servicios de comida en C++
Sistema desarrollado para la gestión integral de pedidos en un servicio de delivery, enfocado en el procesamiento de transacciones y la organización eficiente de múltiples entidades relacionadas.

---

### Funcionalidades Core
* **Administración de Clientes:** Gestión de datos personales, ubicación por distritos e historial operativo.
* **Catálogo de Productos:** Control de platos, categorías y estructuras de precios.
* **Procesamiento de Pedidos:** Registro detallado de transacciones, cálculo automático de subtotales y gestión de cantidades.
* **Generación de Reportes:** Consolidación de información organizada para la toma de decisiones operativa.

### Stack Técnico
* **Lenguaje:** C++ (Programación Modular).
* **Gestión de Datos:** Uso de arreglos estáticos para la administración de estructuras de información.
* **Persistencia:** Manejo de archivos en formato CSV y texto plano para el almacenamiento de registros.
* **Algoritmos:** Implementación de métodos de ordenamiento por nombre/código y algoritmos de búsqueda eficiente en múltiples arreglos.

### Lógica de Ingeniería Aplicada
* **Arquitectura de Software:** Diseño basado en la relación de múltiples entidades (Clientes-Platos-Pedidos).
* **Automatización de Cálculos:** Lógica algorítmica para el procesamiento de montos transaccionales en tiempo real.
* **Optimización de Procesos:** Estructuración de datos complejos para garantizar la integridad de la información durante el flujo de pedidos.

---

### Instalación y Ejecución
1. Clonar el repositorio: `git clone https://github.com/GonzaloLossio/Sistema-Gestion-Pedidos.git`
2. Compilar el proyecto utilizando un compilador estándar de C++.
3. Ejecutar el binario resultante para iniciar la interfaz de consola.
